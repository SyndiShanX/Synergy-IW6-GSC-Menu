# Synergy IW6 GSC Menu

A Ghosts Extinction GSC Mod Menu that uses [IW6-Mod](https://alterware.dev/)

This Menu is a Work-In-Progress, so expect changes and possibly some Bugs

If you use any Code from my Menu, your Menu also has to be Open-Source as per the [GNU GPLv3 License](https://github.com/SyndiShanX/Synergy-IW6-GSC-Menu/blob/main/LICENSE.md)

If you have any Issues/Suggestions, please Submit them in [Issues](https://github.com/SyndiShanX/Synergy-IW6-GSC-Menu/issues)

### Requirements: [IW6-Mod](https://alterware.dev/)

## Installation:

* Download [Synergy.gsc](https://syndishanx.github.io/Synergy-IW6-GSC-Menu/Synergy.gsc)

* Place it into `data\scripts` in the Game Folder
